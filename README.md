# domain-list-china

Domain list based on [felixonmars/dnsmasq-china-list][links-felix-list], Generate by [v2fly/domain-list-community][links-v2fly-list] tools.

[![v2ray-domain-list-china][badge-aur]](links-aur)

## Usage

```json
{
  "routing": {
    "domainStrategy": "AsIs",
    "rules": [
      {
        "type": "field",
        "outboundTag": "direct",
        "domain": [
          "ext:dlc.dat:china",
          "ext:dlc.dat:apple",
          "ext:dlc.dat:google"
        ]
      }
    ]
  }
}
```
